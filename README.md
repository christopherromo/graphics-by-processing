# Graphics by Processing

##### Created by Christopher Romo
##### CS 4800 Computer Graphics : Fall 2024

---

**Graphics by Processing** is a collection of Processing sketches that each showcase an element of art or a principle of design. This project was my introduction to computer graphics, using computer science to create 2D and 3D art. All of the sketches make use of the Processing-specific features, and this project also served as a Java refresher. I used several computer science concepts like data structures and object-oriented programming to explore concepts in the world of visual design.

## Technologies 💻

   - Processing
   - Java

## Features 📄

   - Processing Sketches: Each sketch represents an element of art or principle of design. Other than slight polishing, these remain functionally identical to what I turned in. Each sketch is self-contained.

## Reflection 💭

**The Process:** I started by taking a hour-long course on Processing. I then studied the Processing website as it has documentation and examples for learning. After that came planning, which mostly consisted of physical drawings. The elements of art sketches were created first, being completed in a 1 month period. They cover line, shape, value, color, space, texture, and form, created in that order. The principles of design sketches came next, being completed in a 2 month period. They cover balance, scale, emphasis, proximity & alignment, pattern, contrast, rhythm & flow, harmony, and hierarchy, created in that order. Computer science concepts used to create these sketches include data structures and object-oriented programming. 

**Key Concepts:** This project helped me bridge my two fields of study, computer science and graphic design. At PPSC, I used Adobe Illustrator to create a project about visual design which highlighted the elements of art and principals of design. This project aimed to recreate them using computer science. I went with Java for this project since I knew it on a deeper level than other languages at the time. I learned a lot about computer graphics, specifically how 2D and 3D representations of art are created. For the 2D sketches, I mainly used the Processing-specific features to create interactive art, including functions like draw and mouseClicked. The 3D sketches were similar, but drawing 3D shapes required knowledge of coordinate systems and transformations. Overall, I was able to connect computer science and visual design to create a visually interesting project.

## Running the Project 🎬

1. Clone the repository.

2. Ensure Processing is installed on your computer. 
    - *Made with Processing 4.4.8.*

3. Navigate to the desired sketch.

4. Click the run button in the editor to run the sketch.
    - *Most sketches feature interaction via the mouse!*
